GitHub 認証設定

タスクデータを GitHub リポジトリに保存します。
初回のみ Personal Access Token を設定してください。

  1. GitHub → Fine-grained tokens を開く
  2. Repository: sakura1k83/Kazuya-All-ToDo-data（非公開）を選択
  3. Permissions → Contents: Read and write
  4. Expiration: No expiration を選択
  5. トークンを生成してコピー
